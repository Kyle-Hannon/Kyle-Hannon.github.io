Basis Data Structures

Explaining basic data structures

Stacks

Stacks are linear data structures where insertion and removal is done at the "top" of the stack.

  • To define an empty stack use variable_name = Stack().
  • To check if a stack is empty you use variable_name.isEmpty().
  • To add an item to your stack you use variable_name.push(item)
  • To remove the item on top you use variable_name.pop().
  • To check the number of items in the stack you use vairable_name.size().
  • To get the top item without removing it you use variable_name.peak().

Queues

Queues are linear data structures where insertion is done at one end of the stack and removal is done at the other.

  • To define an empty stack use variable_name = Queue().
  • To chekc if a queue is empty you use variable_name.isEmpty().
  • To add an item to your queue you use variable_name.enqueue(item).
  • To remove the item that is on top of your queue you use variable_name.dequeue().
  • To check the number of items in the queue you use variable_name.size().

Dequeues

Dequeues are linear data structures where insertion and removing can happen at both ends.

  • To define an empty stack use variable_name = Dequeue().
  • To chekc if a dequeue is empty you use variable_name.isEmpty().
  • To add an item to the front of your dequeue you use vairable_name.addFront(item).
  • To add an item to the back of your dequeu you use variable_name.addRear(item).
  • To remove an item from the front of a dequeue you use variable_name.removeFront()
  • To remove an item front the rear of a dequeue you use variable_name.removeRear()
  • To check the number of items in your dequeu you use variable_name.size()
